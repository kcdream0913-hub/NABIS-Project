# BridgeLink — Phase 1 (Atomic Network)

Invite-only professional community for the US–Nepal corridor.

## Run it
```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build (also: npm run verify)
```

## What this build is
The complete Phase 1 dashboard UI with realistic mock data: view system
(US/Nepal/Bridge), feed, community sections, members directory, events + RSVP,
messages, composer, onboarding, profile/settings, and honest locked states for
Phase 3 features. No backend yet — see CLAUDE.md "Next up" for the exact order
(Supabase schema → auth/invites → real data → storage → deploy).
